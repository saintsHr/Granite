# Cloning the Repository

This guide will walk you through getting the official **Granite Engine** source code on your machine. There are two ways to do this:

---

### 1. Downloading the .ZIP File
This is the easiest and fastest method:
1. Go to the official GitHub repository: [https://github.com/saintsHr/Granite](https://github.com/saintsHr/Granite)  
2. Click the green **Code** button: ![Placeholder](../images/codeButton.png)  
3. Select **Download ZIP**: ![Placeholder](../images/downloadButton.png)  
4. Once downloaded, open your file explorer and extract the contents of the ZIP.  
5. Done! You now have the latest source code.

---

### 2. Using Git CLI
This method uses Git to clone the repository directly to your computer:
1. Open your **Terminal** or **Command Prompt**.  

2. Clone the repository:

   ```bash
   git clone https://github.com/saintsHr/Granite.git
   ```

3. Wait for Git to download all files.
4. Done! You now have a folder containing the Granite Engine source code.
